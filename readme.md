A small terminal UI app built with `ratatui` and `crossterm` that renders scrollable text with both vertical and horizontal scrolling. Intended as a base for a terminal reader or for experimenting with scrolling behavior in Ratatui.

---

## Requirements

To use this project, you must have **Rust installed**.

Install Rust using `rustup`:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
Make sure ``cargo`` is available:
cargo --version
## Build & Run
### Run with Cargo (recommended)
cargo run
## Manual build
cargo build
Run the binary:
``./target/debug/elenar2``
For an optimized build:
``cargo build --release
./target/release/elenar2
``
## Notes 
- This project is work in progress.

- Interfaces, behavior, and internal structure may change at any time.

- Intended for learning, experimentation, or as a foundation for a terminal-based reader.
